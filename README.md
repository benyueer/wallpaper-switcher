# Paper Change (壁纸切换器)

一个轻量级的 Windows 壁纸自动切换工具，支持从指定目录加载图片，并通过系统托盘进行管理。

## 功能特性

- **多目录支持**：可以从 `config.txt` 中配置的多个文件夹加载图片。
- **自动循环**：支持每 5 秒自动切换下一张壁纸（可在代码中调整）。
- **手动控制**：通过系统托盘菜单快速切换“上一张”或“下一张”。
- **托盘常驻**：隐藏式运行，提供托盘菜单进行“自动/停止循环”、“打开目录”以及“退出”操作。
- **资源占用极低**：使用 Rust 编写，内存占用极小。

## 快速开始

### 1. 配置图片目录

在程序运行目录下创建或修改 `config.txt`，每一行填写一个图片文件夹的绝对路径：

```text
C:\Users\Username\Pictures\Wallpapers
D:\Images\Nature
```

### 2. 运行

- **开发环境下运行**：
  ```bash
  cargo run
  ```
- **编译发布版**：
  ```bash
  cargo build --release
  ```
  生成的 `.exe` 文件位于 `target/release/paper-change.exe`。

## 托盘菜单说明

右键点击任务栏托盘图标，可以进行以下操作：

- **自动循环**：开启每隔 5 秒自动切换壁纸。
- **停止循环**：停止自动切换。
- **上一张**：手动切换到上一张图片。
- **下一张**：手动切换到下一张图片。
- **打开目录**：在资源管理器中打开 `config.txt` 中配置的第一个目录。
- **退出**：完全关闭程序。

## 技术栈

- **Rust**: 核心逻辑。
- **tray-item**: 实现 Windows 系统托盘功能。
- **wallpaper**: 跨平台壁纸设置库（本项目针对 Windows 优化）。
- **winres**: 处理 Windows 图标资源。

## 注意事项

- 程序目前被硬编码为每 5 秒切换一次（在 `src/main.rs` 的自动循环线程中定义）。
- 请确保 `config.txt` 中的路径存在且包含常见的图片格式。
